## Правило сверки (ADR-006, accepted — заменяет правило ADR-005)

1. **Гранулярность — статья П&Л (line-level).**
2. **Точное совпадение (до копейки) на величинах `ORACLE`.**
3. **Целевая сверка:** Источник #1 (`paymentout`+`cashout`+`loss`, конвертация по курсу документа) + Источник #2 (`entity/commissionreportin`: `reward`÷100 + `commissionOverhead.sum`÷100) → **10 232 903,20**, разрыв закрыт полностью (0,00).
4. **Перемещение исходящий** легитимно исключено из ETL — не участвует в целевой сверке (ADR-005 п.1, подтверждено).
5. **Дашборд-категории (26 шт., по списку заказчика)** — подмножество 31 статьи выше; исключены: Перемещение исходящий, Благотворительность, Возврат займа собственнику, Выплата тела кредита, Проценты по кредиту (движения капитала/финансирования, не операционные расходы); «Комиссия» слита с «Расходы маркетплейсов».
